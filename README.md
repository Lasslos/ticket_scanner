# Ticket Scanner

Ein System, um Tickets zu Veranstaltungen zu erstellen, verwalten und einzulösen.

## Ticket - Datenmodel
Ein Ticket besteht aus folgenden Daten:
- Ticket-ID: Dies ist z. B. die ID des Schülerausweises
- Name: Der Name des Ticketinhabers
- Typ: Der Typ des Tickets (z. B. Vollzahlende, Schüler, ...)
- Kaufzeitpunkt
- Eintrittszeitpunkt
- Notizen: Freitextfeld für Notizen

## Funktionen
- Ticket erstellen
- Ticket bearbeiten
- Eintrittszeitpunkt setzen, d. h. Einlass gewähren
- Ticket löschen
